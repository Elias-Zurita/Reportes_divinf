# Portal de Reportes QA — División Informática · Loan Software

Sitio estático con dos páginas HTML que documentan el proyecto de **generación automatizada de QA con Inteligencia Artificial**, desarrollado en el área de Quality Assurance de División Informática.

## Páginas

| Archivo | Descripción |
|---|---|
| [`qa_reports_portal.html`](./qa_reports_portal.html) | Portal central del área QA — índice de proyectos activos y sus reportes |
| [`reporte_proyecto_ia.html`](./reporte_proyecto_ia.html) | Presentación interactiva del proyecto IA: contexto, metodología, resultados y próximos pasos |

## Estructura

```
ai-project-report/
├── qa_reports_portal.html     # Portal de reportes QA
├── qa_reports_portal.css
├── qa_reports_portal.js

```

## Deploy

El sitio es 100% estático (HTML + CSS + JS vanilla, sin dependencias externas ni build step). 

La página de entrada principal es `qa_reports_portal.html`.
