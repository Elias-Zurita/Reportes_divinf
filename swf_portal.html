Skills Complementarios

Completando el ciclo

🧪
Testing Integracion
Genera tests VB.NET heredando de TestBase. Cubre CRUD completo, casos negativos y validaciones. Lee el documento funcional y crea matriz de cobertura automatica.
👁️
Code Review
Revisa PRs de Gitea validando: nomenclatura (prefijos s, e, o, dt), arquitectura N-Tier, conexiones cerradas en Finally, DataContract/DataMember en entidades.
🔀
Merge Conflicts
Resuelve conflictos analizando git log de ambas ramas y contexto de Jira. Regla critica: nunca usa --ours/--theirs en codigo, solo edita marcadores.
🚀
Cambio Version
Detecta ultima version en Gitea, modifica Comun/Version.vb, hace commit con tag de Jira, push y crea release automaticamente via MCP.

Roadmap

Proximas Mejoras

Evolución planificada del ecosistema de skills de Software Factory.

🏭
Skill Factory
Desarrollo y mantenimiento/evolución de nuevas aplicaciones. Un skill genérico para crear apps desde cero siguiendo los estándares de arquitectura.
💳
Desarrollo-Clear
Skill específico para el desarrollo de Clear, enfocado en personas con conocimiento funcional. Simplifica la curva de aprendizaje técnico.
🔄
Iteración del Skill Desarrollo
Mejora continua del skill principal basado en feedback y casos de uso reales. Optimización de prompts y manejo de edge cases.
🛒
Ecommerce + Servicios Genéricos
Incorporar las apps Ecommerce y Servicios Genéricos al flujo de IA, extendiendo la cobertura del desarrollo asistido.
🔄
AgentSync
Aplicación que distribuye los skills a todos los desarrolladores y los mantiene actualizados automáticamente. Un solo punto de gestión.
